## 🎮 游戏规则

### 基本规则

1. **连接条件**
   - 两个图案必须相同
   - 连接线不能超过两个拐弯
   - 连接路径上不能有其他图案阻挡

2. **消除规则**
   - 点击第一个图案选中
   - 点击第二个相同图案
   - 如果可以连接，则消除
   - 如果不能连接，取消选择

3. **获胜条件**
   - 消除所有图案
   - 在时间限制内完成

### 操作方式

| 操作 | 功能 |
|------|------|
| 点击图案 | 选择/取消选择 |
| 🔄 重新开始 | 重新开始游戏 |
| 💡 提示 | 显示一对可消除的图案 |
| 🔀 重排 | 重新排列剩余图案（扣20分） |

### 计分规则

| 行为 | 得分 |
|------|------|

### 算法实现

#### 路径查找算法

连连看使用BFS（广度优先搜索）算法查找连接路径：

```javascript
// 0个拐弯 - 直接连线
function canConnectDirect(start, end)

// 1个拐弯 - 通过一个中转点
function findPathWithOneTurn(start, end)

// 2个拐弯 - 通过两个中转点
function findPathWithTwoTurns(start, end)
```

#### 检测流程

1. 检查两个图案是否相同
2. 尝试直接连接（0个拐弯）
3. 尝试一个拐弯连接
4. 尝试两个拐弯连接
5. 如果找到路径，绘制连接线并消除

### 数据结构

```javascript
// 游戏棋盘（带边界）
board[rows + 2][cols + 2]

// 边界处理
- 第0行和最后行为边界
- 第0列和最后一列为边界
- 边界值为null，可作为通道
```

## 💡 游戏技巧

### 新手提示

1. **从外向内** - 先消除边缘的图案
2. **使用提示** - 卡住时使用提示功能
3. **及时重排** - 没有可消除对时使用重排
4. **注意时间** - 合理分配时间

### 高级技巧

1. **预判路径** - 提前规划消除顺序
2. **利用边界** - 边界可以作为连接通道
3. **节省提示** - 留着关键时刻使用
4. **快速消除** - 连续消除获得更高效率

### 策略建议

- 🎯 **先易后难** - 先消除明显的对子
- 🔍 **全面扫描** - 定期扫描整个棋盘
- 💡 **合理用提示** - 不要浪费提示机会
- ⏰ **时间管理** - 不要在某个区域停留太久
